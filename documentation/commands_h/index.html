parse

This function is to parse the recived bytes. 

uint32_t parse(
    uint8_t *buffer,
    struct mailbox *ptr);
Parameters
buffer

holds the recived bytes. It is an unsigned char array.

Return Value

1 if it find a valid frame and 0 if not

Discussion

The frame layout: |SOF|message_type|id|message|'\0'|EOF| A message_type character. This tells the recievert which kind of message have to parse. id: which is the id of different messages message: there is a data field end of frame characters:'\0'|EOF Function will search for it and if it finds the function will parse it will parse the frame in to the message structure. And if there is a message it will flush the whole buffer at the end of the parse.
